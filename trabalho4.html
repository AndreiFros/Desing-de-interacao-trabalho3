Painel de dados simultâneos

Esta página utiliza o comando Promise.all para consultar 3 APIs públicas diferentes ao mesmo tempo e exibir os resultados lado a lado. Todas as chamadas são protegidas por try/catch para tratar erros de conexão ou de dados.

As APIs consultadas são:

  • 📍 ViaCEP – Endereço da Avenida Paulista (SP)
  • 🗺 IBGE Localidades – Lista dos estados brasileiros
  • 📰 IBGE Notícias – Últimas notícias do IBGE

Clique em Carregar dados para buscar todas as informações de uma só vez.
