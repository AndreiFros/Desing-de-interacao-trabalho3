Painel de dados simultâneos

Esta página utiliza o comando Promise.all para consultar 3 APIs diferentes ao mesmo tempo e exibir os resultados lado a lado. Todas as chamadas são protegidas por try/catch para garantir que, mesmo que uma API falhe, uma mensagem de erro amigável seja exibida no lugar.

As APIs consultadas são:

  • ⚡ PokeAPI – Informações sobre um Pokémon aleatório
  • 🌤 Open-Meteo – Previsão do tempo em Porto Alegre / RS
  • 🚀 SWAPI – Personagem aleatório do universo Star Wars

Clique em Carregar dados para buscar todas as informações de uma só vez. Cada clique sorteia um Pokémon e um personagem diferente!
