Painel de dados simultâneos

Esta página utiliza o comando Promise.all para consultar 3 APIs diferentes ao mesmo tempo e exibir os resultados lado a lado. Todas as chamadas são protegidas contra erros para garantir que, mesmo que uma API falhe, as demais continuem sendo exibidas.

As APIs consultadas são:

  • 🌍 REST Countries – Informações sobre o Brasil
  • 🚀 Spaceflight News – Última notícia sobre o espaço
  • 💵 AwesomeAPI (Moedas) – Cotação do Dólar (USD) e Euro (EUR)

Clique em Carregar dados para buscar todas as informações de uma só vez.
